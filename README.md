# Projeto labecommerce-backend
<br> Projeto feito no bootcamp de Desenvolvimento Web Full-Stack da Labenu, onde praticamos toda a base de criação de uma API vinculada a um banco de dados real.<br>

## Conteúdos abordados:
- NodeJS
- Typescript
- Express
- SQL e SQLite
- Knex
- Postman
